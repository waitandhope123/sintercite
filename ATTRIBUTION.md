# Attribution Guide

This project is licensed under **Creative Commons Attribution 4.0 International (CC BY 4.0)**.

If you use or adapt material from this repository, please include:

1. **Credit to the original author(s)**  
2. **A link to the Sintercite repository**  
3. **A note describing any modifications (if applicable)**  

### Example Attribution

"Concepts and text adapted from the *Sintercite Hypothesis* project  
(https://github.com/your-username/sintercite), licensed under CC BY 4.0."

You may phrase attribution in any reasonable manner as long as the meaning is preserved.
